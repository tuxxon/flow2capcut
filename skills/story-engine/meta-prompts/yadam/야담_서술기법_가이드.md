# 『야담』 서술 기법 가이드 — Show, Don't Tell & 시네마틱 기법

> 시놉시스·시나리오 작성 시 반드시 참조하는 서술 기법 보조 지침

---

## 1. Show, Don't Tell 감정 처리

### 1.1 기본 원칙

- 감정을 직접 명명하지 않고, **행동·신체 반응·대사·환경**으로 보여준다
- "~했습니다"식 감정 형용사(슬펐습니다, 기뻤습니다, 분노했습니다) 사용을 **최소화**
- 독자가 인물의 상태를 스스로 느끼도록 유도

### 1.2 감정별 신체 반응 매핑

| 감정 | ❌ Tell (지양) | ⭕ Show (권장) |
|------|---------------|---------------|
| 슬픔 | "슬펐습니다" | 고개를 숙이다, 입술을 깨물다, 눈시울이 붉어지다 |
| 분노 | "분노했습니다" | 주먹을 불끈 쥐다, 이를 악물다, 목소리가 떨리다 |
| 공포 | "두려웠습니다" | 등줄기에 한기가 흐르다, 다리가 굳다, 숨을 삼키다 |
| 사랑/설렘 | "사랑했습니다" | 심장이 빠르게 뛰다, 볼이 붉어지다, 눈을 피하다 |
| 기쁨 | "기뻤습니다" | 입꼬리가 올라가다, 발걸음이 가벼워지다, 콧노래를 흥얼거리다 |
| 긴장 | "긴장했습니다" | 손에 땀이 배다, 침을 삼키다, 시선이 흔들리다 |
| 후회 | "후회했습니다" | 한숨을 내쉬다, 빈 손을 쥐었다 폈다 하다, 먼 곳을 응시하다 |
| 감동 | "감동했습니다" | 눈시울이 뜨거워지다, 목이 메다, 말없이 고개를 끄덕이다 |
| 수치심 | "부끄러웠습니다" | 고개를 돌리다, 얼굴이 달아오르다, 손으로 얼굴을 가리다 |
| 그리움 | "그리웠습니다" | 먼 하늘을 올려다보다, 낡은 물건을 어루만지다, 이름을 되뇌다 |

### 1.3 Show 기법 세부 전략

| 기법 | 설명 | 예시 |
|------|------|------|
| **행동으로 보여주기** | 감정 대신 인물의 구체적 행동 묘사 | "서리는 몸을 돌려 눈물을 감추었습니다." |
| **신체 반응** | 무의식적 신체 변화로 감정 암시 | "선비의 손끝이 미세하게 떨리고 있었습니다." |
| **환경 반영** | 주변 환경이 인물의 감정을 대변 | "먹구름이 몰려오더니 빗방울이 떨어지기 시작했습니다." |
| **대사의 행간** | 말하지 않는 것, 말을 돌리는 것으로 표현 | "괜찮소." 선비는 고개를 돌리며 짧게 내뱉었습니다. |
| **사물/소품 활용** | 특정 사물에 감정을 투영 | "서리는 떨어진 꽃잎을 한참이나 내려다보았습니다." |
| **침묵과 여백** | 말없는 순간으로 감정의 무게를 전달 | "아무 말도 하지 않았습니다. 다만 길게 내쉬는 숨소리만이 방 안을 채웠지요." |
| **반복 행동** | 불안·초조 등을 반복 동작으로 표현 | "선비는 자꾸만 갓끈을 매만졌습니다." |
| **대조/역설** | 겉과 속이 다른 행동으로 복잡한 감정 표현 | "입가에는 미소를 띠었지만, 소매 안에 감춘 두 손은 바르르 떨고 있었습니다." |

### 1.4 변환 예시 모음

**예시 1 — 감동**
> ❌ Tell: "서리는 깊이 감동했습니다."
>
> ⭕ Show: "서리의 눈시울이 붉어지며 입술이 파르르 떨렸습니다. 한참을 말없이 서 있던 서리는 고개를 깊이 숙였지요."

**예시 2 — 분노**
> ❌ Tell: "선비는 크게 분노했습니다."
>
> ⭕ Show: "선비의 이가 딱 하고 맞물리는 소리가 들렸습니다. 불끈 쥔 주먹에서 핏줄이 도드라졌지요."

**예시 3 — 사랑**
> ❌ Tell: "두 사람은 서로를 사랑했습니다."
>
> ⭕ Show: "서리의 시선이 선비의 뒷모습에 닿자, 자신도 모르게 입가에 엷은 미소가 번졌습니다. 귓불이 붉어지는 것은 찬바람 탓이라 여기며 고개를 돌렸지요."

**예시 4 — 공포**
> ❌ Tell: "그녀는 두려웠습니다."
>
> ⭕ Show: "서리의 등줄기를 타고 차가운 것이 한 줄기 흘러내렸습니다. 다리에 힘이 빠져 그 자리에 주저앉고 말았지요."

## 2. 시네마틱 서술 기법

### 2.1 카메라 워크적 묘사

장면을 영상처럼 구성하여 독자의 머릿속에 화면이 그려지도록 한다.

| 기법 | 설명 | 예시 |
|------|------|------|
| **와이드 샷** | 배경과 분위기를 넓게 잡아 시작 | "드넓은 들판 너머로 한양 성곽이 아스라이 보였습니다." |
| **미디엄 샷** | 인물의 행동과 대화 중심 묘사 | "선비는 붓을 내려놓고 창밖을 바라보았습니다." |
| **클로즈업** | 표정, 손짓, 사물의 디테일에 집중 | "그의 손가락 끝에 묻은 먹물이 종이 위로 한 방울 떨어졌습니다." |
| **익스트림 클로즈업** | 극도로 작은 디테일로 긴장감 극대화 | "떨리는 눈동자 속에 촛불의 그림자가 일렁이고 있었습니다." |
| **주관 시점 (POV)** | 특정 인물의 시선으로 장면 묘사 | "서리의 눈에 비친 것은 달빛 아래 홀로 서 있는 선비의 뒷모습이었습니다." |

**활용 원칙:**
- 장면 시작 시 **와이드 샷**으로 공간감을 잡고, 점차 **클로즈업**으로 좁혀간다
- 감정이 고조되는 순간에는 **클로즈업**으로 집중한다
- 인물의 심리를 드러낼 때 **POV**를 활용한다
- 반대로 **클로즈업 → 와이드 샷** 전환은 인물의 고립감이나 무력감을 표현할 때 효과적

### 2.2 씬 전환 기법

| 기법 | 설명 | 예시 |
|------|------|------|
| **하드 컷** | 장면을 즉시 전환 (단락 분리) | 한 장면 끝 → 새 단락으로 바로 다음 장면 시작 |
| **디졸브** | 현재 이미지가 다음 장면과 겹치듯 전환 | "흩날리는 꽃잎이 어느새 눈송이로 바뀌어 있었습니다." |
| **페이드 아웃/인** | 어둠이나 빛으로 장면 전환 | "어둠이 내려앉았습니다. 이윽고 동이 트자..." |
| **매치 컷** | 유사한 이미지나 동작으로 연결 | "찻잔에 비친 둥근 달 → 하늘의 보름달로 시선이 옮겨갔습니다." |
| **사운드 브릿지** | 소리로 장면을 이어 붙임 | "북소리가 울려 퍼지는가 싶더니, 이내 그것은 빗줄기가 지붕을 두드리는 소리였습니다." |

### 2.3 시간 처리 기법

| 기법 | 설명 | 예시 |
|------|------|------|
| **몽타주** | 시간 경과를 압축된 장면들로 보여줌 | "봄꽃이 피고 지고, 매미 소리가 울리고, 낙엽이 지는 동안 선비는 한 번도 붓을 놓지 않았습니다." |
| **슬로모션** | 결정적 순간을 느리게, 세밀하게 묘사 | 한 순간을 여러 문장에 걸쳐 감각별로 분해 묘사 |
| **플래시백** | 과거 회상을 통한 정보 전달 | "문득 어린 시절의 기억이 스쳐 지나갔습니다. 그날도 이렇게 비가 내리고 있었지요." |
| **타임 스킵** | 시간 도약으로 불필요한 구간 생략 | "며칠이 흘렀습니다." / "봄이 가고 여름이 왔습니다." |

### 2.4 병렬 편집 (교차 편집)

두 장면을 **교차**하며 긴장감을 조성하는 기법이다.

- "그 시각, 다른 곳에서는..."
- 위기 장면과 구원 장면을 번갈아 배치하여 서스펜스 극대화
- 두 인물의 행동을 교차 묘사하여 충돌의 순간을 예감하게 함

**예시:**
> "선비가 칼을 뽑아 드는 그 순간, 서리는 언덕 너머를 향해 필사적으로 달리고 있었습니다. 선비의 칼끝이 허공을 갈랐습니다. 그리고 그 칼이 내리꽂히려는 바로 그때, 멀리서 서리의 외침이 산을 넘어 울려 퍼졌지요."

### 2.5 오감 기반 장면 구축

장면을 구성할 때 **최소 2~3가지 감각**을 동원하여 몰입감을 높인다.

| 감각 | 활용 방법 | 예시 |
|------|-----------|------|
| **시각** | 색감, 빛, 그림자, 움직임 | "붉은 노을이 방 안까지 스며들었습니다." |
| **청각** | 소리, 침묵, 울림 | "풀벌레 소리만이 고요한 밤을 채우고 있었지요." |
| **촉각** | 온도, 질감, 바람 | "차가운 바닥이 맨발의 온기를 빼앗아 갔습니다." |
| **후각** | 향기, 냄새 | "은은한 매화 향이 바람결에 실려 왔습니다." |
| **미각** | 맛, 음식 장면 | "쓰디쓴 약초 맛이 혀끝에 오래도록 남았지요." |

**장면 구축 예시 (오감 종합):**
> "차가운 밤바람이 옷깃을 파고들었습니다(촉각). 멀리서 풀벌레 소리가 가늘게 들려오고(청각), 달빛에 젖은 풀잎이 은빛으로 반짝이고 있었지요(시각). 서리는 품속에서 꺼낸 떡을 한 입 베어 물었습니다(미각). 솔잎 위에 맺힌 이슬의 향기가 코끝을 스쳤지요(후각)."

## 3. 시놉시스 단계 적용

시놉시스 작성 시에도 Show/시네마틱 기법을 **설계 수준**에서 미리 준비한다.

### 3.1 감정 설계

| 규칙 | 설명 |
|------|------|
| **감정-행동 매핑** | 주요 감정 장면마다 "어떤 행동/반응으로 보여줄 것인지" 미리 기획 |
| **환경-감정 연결** | 핵심 장면의 배경(날씨, 계절, 시간대)을 감정선과 연동 설계 |
| **상징물 배치** | 감정을 대변할 소품이나 자연물을 시놉시스 단계에서 설정 |

### 3.2 시네마틱 장면 설계

| 규칙 | 설명 |
|------|------|
| **핵심 장면 시점 설계** | 중요 장면의 시작 시점(와이드/클로즈업 등) 미리 구상 |
| **씬 전환 포인트** | 챕터 간 전환 방식(컷/디졸브/페이드) 미리 설정 |
| **슬로모션 지점** | 절정/반전 장면에서 시간을 늘려 묘사할 지점 표시 |
| **병렬 편집 장면** | 교차 편집이 필요한 장면을 시놉시스에서 미리 구성 |
