> 작성 중에도 매 챕터 전환 시 "챕터 전환 체크포인트"를 반드시 점검한다.

## 🔴 절대 금지 사항 (가장 자주 실수하는 항목)

| # | 금지 사항 | 올바른 표현 |
|---|----------|------------|
| 1 | 감정 직접 명명 ("슬펐습니다", "기뻤습니다", "분노했습니다") | 행동/신체반응으로 **Show** |
| 2 | 말줄임표 (.., …) | 문장을 끝까지 완결 |
| 3 | 외래어 (키스, 다크서클 등) | 입맞춤, 눈가의 그늘 |
| 4 | 영어/알파벳 일체 | 한글로 대체 |
| 5 | 숫자 아라비아 표기 (8시, 3명) | 한글 표기 (여덟 시, 세 명) |
| 6 | '그', '그녀' 지칭 | 이름이나 호칭 사용 |
| 7 | "~씨" 호칭 | 낭자, 도령, 마님, 서방님 등 |
| 8 | "하녀" 사용 | 몸종, 침모 |

## 🟡 핵심 기법 확인

### Show, Don't Tell

작성 중 아래 질문을 스스로에게 던진다:

- [ ] 감정을 **행동/신체반응/환경/대사**로 보여주고 있는가?
- [ ] 감정 형용사("~했습니다") 직접 사용을 최소화했는가?
- [ ] 인물의 심리 상태를 **소품이나 자연 현상**에 투영했는가?
- [ ] **대사의 행간** — 말하지 않는 것으로 감정을 전달하고 있는가?

**빠른 참조:**

| 이렇게 쓰면 안 됨 | 이렇게 바꿔라 |
|-------------------|-------------|
| "서리는 깊이 감동했습니다." | "서리의 눈시울이 붉어지며 입술이 파르르 떨렸습니다." |
| "선비는 크게 분노했습니다." | "선비의 이가 딱 하고 맞물리는 소리가 들렸습니다." |
| "두 사람은 서로를 사랑했습니다." | "서리의 귓불이 붉어지는 것은 찬바람 탓이라 여기며 고개를 돌렸지요." |
| "그녀는 두려웠습니다." | "서리의 등줄기를 타고 차가운 것이 한 줄기 흘러내렸습니다." |

### 시네마틱 기법

- [ ] 장면 시작 시 **와이드→클로즈업** 시점 변화를 활용했는가?
- [ ] 장면 전환이 자연스럽게 연결되는가? (컷/디졸브/페이드/매치컷/사운드브릿지)
- [ ] 절정 장면에서 **슬로모션** 기법을 활용했는가?
- [ ] 필요 시 **병렬 편집**(교차 장면)을 활용했는가?
- [ ] **오감 중 최소 2~3가지**를 동원하여 장면을 구축했는가?

### 서술 기본

- [ ] 3인칭 관찰자 시점 일관성
- [ ] "~습니다", "~지요" 종결형
- [ ] "옛날 옛적" 또는 "먼 옛날"로 시작
- [ ] 마무리 CTA 멘트 포함
- [ ] 신분에 따른 적절한 화법 (양반↔서민 존비어)

## 🟢 챕터 전환 체크포인트

**매 챕터 완성 후 아래 항목을 반드시 점검한다:**

### 서술 품질
- [ ] 이 챕터에서 감정을 직접 명명한 곳이 없는가? (Show, Don't Tell)
- [ ] 최소 1개 이상의 시네마틱 기법(카메라워크/씬전환/시간처리)을 활용했는가?
- [ ] 오감 중 2가지 이상을 활용한 장면이 있는가?

### 구조 일관성
- [ ] 이전 챕터와 자연스럽게 연결되는가?
- [ ] 배분된 글자 수를 준수했는가?
- [ ] 인물 설정(신분, 호칭, 말투)이 일관적인가?
- [ ] 외래어/현대어가 섞이지 않았는가?

### 복선 관리
- [ ] 이 챕터에서 설치해야 할 복선이 배치되었는가?
- [ ] 이 챕터에서 회수해야 할 복선이 회수되었는가?
- [ ] 아래 복선 추적표가 최신 상태인가?

## 📐 글자 수 배분 빠른 참조 (시나리오용)

| 구간 | 챕터 | 글자 수 |
|------|------|---------|
| 【기】 일상과 욕망 | 1~5 | 1,500자 |
| 【승】 관계 심화와 균열 | 6~11 | 3,000자 |
| 【전】 위기와 배신 | 12~17 | 3,500자 |
| 【결】 반전과 해결 | 18~20 | 2,000자 |
| **합계** | **20챕터** | **10,000자** |

> **적정 범위:** 8,000자 ~ 12,000자 (너무 길면 영상 개발 시간이 과도하게 늘어남)

> **글자 수 측정 기준:** `LC_ALL=ko_KR.UTF-8 wc -m` (UTF-8 한글 기준, 바이트가 아닌 문자 수)

## 🔄 작업 흐름

```
1. 이 프리플라이트 문서 읽기
2. 복선 추적표 초안 작성
3. 챕터 1 작성 시작
4. 챕터 완성 → "챕터 전환 체크포인트" 점검
5. 확인 요청: "이것으로 [X]챕터가 완성되었습니다. 다음 챕터로 진행해도 될까요?"
6. 승인 후 다음 챕터 → 4번으로 돌아감
7. 전체 완성 후 종합 체크리스트 최종 점검
```
