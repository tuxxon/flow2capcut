## 참고: 스하리란?

- **스하리**: 스레드(팔로우) + 하트(좋아요) + 리포스트 = 세 가지를 한꺼번에 하는 것
- **반하리**: 스하리를 받으면 되돌려주는 것 (반사 스하리)
- **또하리**: 이미 스하리한 글을 한 번 더 스하리
