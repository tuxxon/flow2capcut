## 2. 알고리즘 랭킹 가중치

| 요소 | 비중 | 설명 |
|------|------|------|
| **Engagement Velocity** | **40%** | 게시 후 30~90분 내 참여 속도. 30분 내 답글 50개 > 24시간 좋아요 200개 |
| **Completion Rate** | **25%** | 끝까지 읽었는지 + 저장(bookmark) 여부 |
| **Conversation Quality** | **20%** | 성의있는 답글 > 단순 좋아요. 작성자가 답글에 참여하면 부스트 |
| **Network Relevance** | **10%** | 비슷한 관심사 유저에게 노출 |
| **Freshness** | **5%** | 꾸준한 게시가 기본 도달 올림 |

> 출처: MomentumHive 알고리즘 가이드 (2026)

### 알고리즘이 싫어하는 것
- "Like if you agree" 류 engagement bait → **패널티**
- 본문에 외부 링크 → 도달 감소 (Mosseri가 최근 완화했다고 하지만 여전히 주의)
- **남의 글** 복사/붙여넣기 → 독창성 부족으로 비추천
  - 단, **자기 글 재게시**는 OK — 1~2주 간격, 훅/이미지 등 약간 변형하면 더 효과적
  - 새 팔로워는 이전 글을 못 봤으므로 잘 된 글은 반복 재활용 권장
- 비일관적 게시 패턴 → 기본 도달 하락

### Instagram 연동 효과
- Instagram에서 프로필 방문, 게시물 참여 → Threads "For You" 피드에 영향
- **Account Status 대시보드**에서 추천 자격 여부 확인 가능

## 3. 콘텐츠 포맷별 성과 데이터

| 포맷 | 참여율 비교 | 비고 |
|------|------------|------|
| **이미지 포함** | 텍스트만 대비 **+60~64%** | 4:5 세로형 권장 |
| **30초 미만 동영상 + 자막** | 조회수 **+48%** | 가장 높은 조회수 |
| **캐러셀 (여러 이미지)** | 좋아요 **2배** | 첫 이미지 + 한 줄 훅이 핵심 |
| **동영상 답글** | 세션 시간 **+30%** | |
| **1인칭("I") 글** | 3인칭 대비 **+23%** | |
| **1~2문장 단락 + 줄바꿈** | 밀집 텍스트 대비 **2.3배** | 모바일 가독성 |
| **3~4문단마다 이미지** | 완독률 **+45%** | |
| **텍스트만** | 기준 | 최하위 성과 |

## 5. 인디해커/개발자 제품 홍보 DO/DON'T

### DO
- 개발 과정/삽질 스토리 공유 (authentic)
- 질문형으로 시작해서 대화 유도
- **답글을 게시물보다 더 많이 달기** (Adam Mosseri 직접 발언)
- 이미지/스크린샷 포함 (4:5 세로형)
- 토픽 태그 1개 사용 (해시태그 아님)
- 첫 100자에 훅(hook) 넣기 → 초기 조회수 **+40%**
- 주 3~5회 일관된 게시
- 인기 글에 가치 있는 답글 달기 → 그 글의 청중을 빌려옴
- 스크린샷에 핸들(@) 포함 → 공유될 때 내 계정 노출

### DON'T
- 첫 글에 링크 넣기 (계정 삭제 위험)
- 팔로워 300명 미만에서 남의 글에 댓글 달기
- 좋아요/팔로우 직접 요청 → 패널티
- 광고/보도자료 스타일
- 해시태그 남발
- 완벽하게 다듬은 마케팅 문구
- 과도한 게시 (낮은 참여율 글이 쌓이면 향후 글도 억제)
